# Contributing

Thank you for your interest in Skill Studio. Contributions are welcome.

## Requirements

- Node.js 20+

## Setup

```bash
git clone https://github.com/uparogya/vscode-skill-studio.git
cd vscode-skill-studio
npm install
```

## Development

Press **F5** in VS Code to launch an Extension Development Host with the extension loaded. The `launch.json` and `tasks.json` in `.vscode/` handle compilation automatically.

For continuous compilation in a terminal:

```bash
npm run watch
```

## Submitting changes

1. Fork the repository and create a branch from `main`.
2. Make your changes and verify the extension works end-to-end in the Extension Development Host.
3. Keep commits focused — one logical change per commit.
4. Open a pull request with a clear description of what changed and why.

## Reporting bugs

Open an issue at [github.com/uparogya/vscode-skill-studio/issues](https://github.com/uparogya/vscode-skill-studio/issues). Include your OS, VS Code version, and steps to reproduce.
